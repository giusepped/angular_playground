# Angular playground

In this repo, I just play around with angularJS and testing in order to learn more about it.


# sw-front

version 0.14.0.

## Build & development

Run `grunt` for building and `grunt serve` for preview.

## Testing

Running `grunt test` will run the unit tests with karma.
